# ToyProject-Board-App  📝🖊
## 🖥 Project Summary
- 게시판(board)과 사용자(user) App을 구현했습니다.

## 👩🏻‍💻 Work Details
- 다음과 같은 기능을 구현했습니다.
   - 유저 회원가입
   - 유저 로그인 
   - 작성자 글 작성
   - 작성자 글 수정 (작성자 비밀번호로 수정)
   - 작성자 글 삭제 (작성자 비밀번호로 삭제)
   - 게시판 글 목록
   - 게시판 글 확인
   - Pagination
   - 답글 기능
   - 게시글 작성자 IP 표시
   - Unit test
